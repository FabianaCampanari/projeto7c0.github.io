  • Código Python

    O projeto 7c0 é escrito na linguagem Python, na versão 3.7.2. Além disso, utilizamos diversas bibliotecas para facilitar o desenvolvimento: PyMySql para conexão com o banco de dados; Tweepy para conexão com as APIs do Twitter archiveis para realizar o arquivo dos tweets; E pandas para manipulação de arquivos do tipo .csv.

  • Lista de políticos

    A lista de políticos utilizada pelo projeto 7c0 está publicada no Github do projeto, e pode ser utilizada por qualquer pessoa interessada nessas informações. Caso perceba algum erro na lista, nos avise que atualizamos.

  • Banco de dados

    O projeto armazena em um banco de dados MySQL as seguintes informações relativas as postagens no Twitter: identificador da postagem, nome do usuário que realizou a postagem, data e hora, o conteúdo postado, e um link da postagem arquivada no site archive.vn.

  • As APIs do Twitter

    O projeto utiliza as APIs gratuitas do Twitter, que mesmo com suas limitações, permite a captura de dados no volume atual do projeto 7c0, com mais de 400 contas de políticos brasileiros ativos.

  • Contribua no GitHub

    Convidamos você para contribuir no nosso github de duas formas: Ajudando a manter a lista de políticos e suas redes sociais atualizada e sugerindo melhorias no código-fonte do projeto.
