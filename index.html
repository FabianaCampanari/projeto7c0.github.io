O que é o projeto?

Não podemos nos esquecer das promessas que os políticos fazem, nem tão pouco permitir que deletem palavras ditas publicamente. O 7C0 faz esse papel no Twitter.

Siga no Twitter

A conta @projeto7C0 no Twitter posta todos os dias tweets que foram apagados por diversos políticos brasileiros. Siga a gente lá!

Github do Projeto

O código-fonte do projeto, a lista de políticos acompanhados e o código-fonte deste site estão disponíveis no Github.
